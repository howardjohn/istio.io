We recommend using a version of Kubernetes that has been tested with Istio:

|Istio Version       | Kubernetes Version
|--------------------|-----------------------
|1.6                 | 1.15, 1.16, 1.17, 1.18
|1.5                 | 1.14, 1.15, 1.16
|1.4                 | 1.13, 1.14, 1.15

For more information, see the [Versioning Policy](/about/version-policy/).
